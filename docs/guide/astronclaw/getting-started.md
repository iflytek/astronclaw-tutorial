# 快速上手指南

只需几个简单的步骤，即可拥有一只属于你的、全天候在线的云端数字龙虾。

## 第一步：立即部署

进入 **AstronClaw** 页面，点击 `立即部署` 按钮，系统将自动为你创建 AstronClaw。

## 第二步：开始对话

部署完成后进入对话首页：
*   **欢迎语**：Hi，我是 AstronClaw！
*   **文件上传**：
    *   支持 `pdf/doc/txt/jpg` 等格式
    *   文件大小不超过 50MB，总文件大小不超过 250MB，数量不超过 5 个
    *   音频文件大小不超过 100MB，总文件大小不超过 500MB，数量不超过 5 个
    *   图片大小不超过 20MB，总文件大小不超过 100MB，数量不超过 5 个

## 第三步：安装技能

进入 **Skills市场**，浏览和安装官方预置技能：
*   支持名称模糊搜索
*   点击安装即可一键配置使用

*(更多技能相关的详细介绍，请参考 [技能系统详解](./skills.md))*

## 第四步：配置消息渠道

进入 **渠道配置** 页面，连接企业协作平台。AstronClaw 现已支持：
*   飞书渠道
*   钉钉渠道
*   企业微信渠道

只需简单配置相关 App ID 和凭据，你的数字龙虾即可在这些聊天软件中随时待命！
