# 积分体系与订阅

## 1. 规则说明
本规则适用于 AstronClaw 服务中的订阅套餐、积分获取、积分使用及计费机制。 
AstronClaw 使用**积分（Points）**作为统一计费单位，用于衡量用户在平台上的： 
*   计算资源消耗 
*   大模型调用 
*   Skill能力调用 
*   Agent任务执行

## 3. 积分类型
AstronClaw 平台积分分为三种类型，不同积分的有效期不同。 

| 积分类型 | 有效期 | 说明 |
| :--- | :--- | :--- |
| **会员积分** | 当月有效 | 随订阅套餐赠送，不可累积到下个月 |
| **增购积分** | 购买日起 1年有效 | 用户额外购买的积分 |
| **活动积分** | 以活动规则为准 | 参与官方活动获得 |

**说明：**
*   会员积分为**非自然月重置** 
*   若会员终止，未使用积分将自动失效

## 12. 积分退款政策
### 异常退回 
若任务执行出现以下情况： 
*   系统错误 
*   服务中断 
*   审核失败 
系统将**自动退还全部积分**。 

### 不可退款 
已购买：套餐、积分包 
不支持提现或转让。 

### 作弊处理 
若发现以下行为： 
*   利用漏洞刷积分 
*   非法调用API 
平台有权： 
*   清零积分 
*   封禁账户 

## 13. 常见问题（FAQ）
**Q1：当月积分用完怎么办？**
A：可以购买积分包、提前续费或升级套餐。

**Q2：系统优先扣除哪种积分？**
A：系统优先扣除即将过期的积分（通常为会员积分）。

**Q3：可以同时使用多个套餐吗？**
A：不可以。每个账户仅可订阅一个套餐。

**Q4：如何查看积分使用情况？**
A：可在积分管理页面查看剩余积分、积分明细和购买记录。
