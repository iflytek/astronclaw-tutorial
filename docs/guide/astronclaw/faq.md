# 常见问题

## 一键部署需要什么条件？ 
需要订阅基础版或专业版套餐。未订阅用户点击部署时会引导升级。 

## 如果 AstronClaw 不响应怎么办？ 
*   刷新页面 
*   进入设置点击 **重启AstronClaw** 
*   如仍无效，尝试设置中的自动修复 

## 重启、重置、删除有什么区别？ 

| 操作 | 效果 |
| :--- | :--- |
| **重启** | 保留数据，清空缓存（类似重启电脑） |
| **重置** | 删除历史记录和技能，恢复默认设置（类似重装系统） |
| **删除** | 彻底删除 AstronClaw 及所有数据（不可恢复） |

## 支持哪些文件上传？ 
*   **支持格式**：pdf、doc、txt、jpg 等，类型不限 
*   **普通文件**：每个文件大小不超过 50MB，总文件大小不超过 250MB，数量不超过 5 个 
*   **音频文件**：不超过 100MB，总文件大小不超过 500MB，数量不超过 5 个 
*   **图片文件**：不超过 20MB，总文件大小不超过 100MB，数量不超过 5 个 

## 如何切换对话模型？ 
进入 **设置** 页面，在模型选择区域切换对话模型。 

## AstronClaw 可以发送文件给我吗？ 
目前 AstronClaw 可以接收你上传的文件并用工具处理，但尚不能在界面中直接发送文件给你。该功能将在未来版本中推出。现阶段，你可以通过其他已配置的渠道访问生成的文件。
