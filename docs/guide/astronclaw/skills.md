# 技能系统 (Skills)

技能（Skills）是 AstronClaw 的核心能力模块，相当于 Agent 可调用的“工具与能力插件”。通过技能，AstronClaw 可以从简单的“聊天”升级为真正的“执行任务”。

## 一、什么是技能？

**技能 = 可被 AstronClaw 调用的能力单元**，包括：
*   外部工具（搜索 / OCR / API）
*   模型能力（语音 / 图像 / 推理）
*   工作流（多步骤自动执行任务）
*   行业模板（研报 / PPT / 分析报告）

## 二、技能如何获取

AstronClaw 支持 3 种技能获取方式：

### 1. 技能市场手动安装
进入【Skills市场】：
*   支持关键词搜索（如：PPT / 搜索 / OCR）
*   浏览官方推荐技能
*   点击「安装」即可使用
> **特点**：稳定、官方验证；适合已知需求场景。

### 2. 仓库搜索安装
AstronClaw 支持从技能仓库进行扩展：
*   搜索技能（官方 / 第三方）
*   浏览技能说明
*   手动安装到「我的技能」
> **适用**：寻找特定能力（如行业工具、垂类能力）；扩展生态能力边界。

### 3. 对话自动安装 (推荐)
用户可以直接在对话中描述需求，让 AstronClaw 自动完成技能获取与安装：
> **系统行为**：自动理解需求 → 在技能市场/仓库中搜索匹配技能 → 自动完成安装 → 立即可用
> **特点**：无需进入技能市场，无需手动操作，完整自动化闭环。

## 三、技能如何被调用

AstronClaw 支持 3 种调用方式：

1.  **自动调用**：用户自然表达需求，系统自动调用相关技能。
2.  **指定调用**：用户明确指令让 AstronClaw 执行某个技能。
3.  **多技能链式调用**：系统自动完成连续任务，例如：`搜索` → `分析` → `图表` → `PPT生成`。

## 四、技能市场与自定义

当前技能市场（Skills Market）提供 **130+** 官方技能，持续扩展中：
*   **数据分析**（CSV / 图表）
*   **内容生成**（PPT / 文档）
*   **信息获取**（搜索 / 资讯）
*   **AI能力**（OCR / 语音）

### 自定义技能
AstronClaw 支持用户创建专属技能：
*   **上传创建**：支持上传 `.zip` / `.md` 文件自动生成技能
*   **对话创建**：直接在与 AstronClaw 对话中描述需求，AI 将为你编写和生成专属技能配置。

### 我的技能（管理）
在“我的技能”面板，你可以：
*   **新建**：上传文件创建
*   **删除**：移除技能
*   **下载**：导出技能
*   **启用/禁用**：控制技能调用状态
