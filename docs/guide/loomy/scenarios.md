# 典型场景

Loomy 适合那些需要频繁在信息整理、内容创作、任务执行和跨工具协作之间切换的工作场景，尤其适合个人工作者、小团队以及高频办公用户。

以下是 Loomy 的几个典型落地场景：

## 1. 晨间智能播报
想要快速了解今天的重要信息？Loomy 可以帮你整合天气、邮件、日程和热点资讯，生成一份专属晨间播报，让你从容开启新的一天。

**使用场景**
> 今天周一，帮我看看今天合肥天气适不适合出去玩，有什么有意思的新闻，邮箱里有没有需要回的东西，做成一张卡片给我

Loomy 会帮你：
*   **天气播报**：查询实时天气，分析是否适合户外活动，智能推荐穿搭建议
*   **新闻摘要**：精选当日热点和有趣资讯，3分钟掌握关键信息
*   **邮件速览**：扫描未读邮件，标注重要邮件和待回复事项
*   **卡片生成**：将所有信息整合成一张精美的晨间播报卡片

![晨间智能播报](/loomy/scenarios/case1.png)

## 2. 学习进度管理
需要复盘学习进度？Loomy 可以帮你分析执行情况，找出问题所在，并给出更现实的调整建议。

**使用场景**
> 这周是我备考 CPA 会计科目的第三周，完成情况是这样的：
>
> 周一到周三：按计划学完了金融资产章节 周四：加班没学 周五：只做了一半的练习题 周末：补了周四的进度，但周五的题还没做完
>
> 帮我复盘一下这周的执行率，分析一下我容易断在哪，下周的计划怎么调整能更现实一点。

Loomy 会为你：
*   **执行率分析**：计算本周完成度，识别计划与实际的差距
*   **断点诊断**：分析容易中断学习的时间节点和原因
*   **计划优化**：基于实际情况，调整下周计划的节奏和任务量
*   **提醒建议**：给出关键时间点的提醒建议，保持学习连续性

![学习进度管理](/loomy/scenarios/case4.png)

## 3. 内容创作与运营
从选题策划到内容发布，再到数据复盘，Loomy 串联选题、热点分析、内容创作与分发，让自媒体创作更快进入产出状态。

**使用场景**
> 我运营一个抖音账号，定位是科技数码测评。最近流量有点下滑，帮我分析一下这周科技圈有哪些热门话题，给我推荐 3 个选题方向，每个方向给出标题、核心卖点和预估的受众反应。

Loomy 会帮你：
*   **热点追踪**：实时抓取圈内热门话题，分析讨论热度和趋势走向
*   **选题推荐**：结合账号定位，推荐高潜力选题方向
*   **初稿生成**：选定选题后，快速生成视频脚本或图文初稿

![内容创作与运营](/loomy/scenarios/case2.png)

## 4. 文档智能管理
Loomy 打通文件整理、知识沉淀与内容输出环节，把碎片化工作变成可复用的高效流程。

**使用场景**
> 我下载文件夹里堆了一堆乱七八糟的文件，还有几个随手存的截图。帮我按类型和项目自动分类归档，建好文件夹结构，告诉我你是怎么归档的。

Loomy 会为你：
*   **智能识别**：识别文件类型（文档、图片、表格、PDF等）
*   **结构化归档**：创建清晰的文件夹层级，按"项目/类型/日期"自动归档
*   **归档报告**：生成归档说明，告诉你每个文件被放到了哪里，方便后续查找

## 5. 电商直播助手
Loomy 帮助电商从业者快速获取商品信息、分析市场数据，并生成专业的直播话术和营销文案。

**使用场景**
> 打开京东，把首页的前十条商品以及他的设计文案、订单销量等信息保存到本地，假设我现在直播要推广这十个商品，我应该如何设计直播话术？

Loomy 会帮你：
*   **商品信息抓取**：提取商品页面前十条商品的详细信息
*   **卖点提炼**：分析商品优势，提炼核心卖点，突出性价比和用户痛点
*   **话术生成**：基于商品特点和销量数据，为每个商品生成吸引人的直播话术
