# 常见问题 (FAQ)

## 隐私与安全相关

### 我的文件会被上传到云端吗？

Loomy 的文件操作默认在本地完成，不会直接把你的整个工作目录上传到云端。但为了实现 AI 理解、总结和分析，相关文本内容会发送给大模型 API 服务商进行处理。

### AI 能访问我的哪些文件？

Loomy 只能访问你**明确授权的工作目录**。对于未授权目录中的文件，如果任务执行过程中确实需要访问，Loomy 会先征求你的同意，不会默认直接读取。

### Loomy 可以使用我自己的模型服务提供商 API Key 吗？

可以。Loomy 提供默认的模型服务，方便用户开箱即用；如果你有自己的模型服务提供商，也可以在 Loomy 中自行配置 API Key 进行使用。
当你使用自己的 API Key 时，Loomy 不会额外上传任何涉密信息。相关 Key 只会保存在本地，并在你发起对应模型调用时被本地使用，不会被 Loomy 平台托管或同步到云端。

## 适用人群

### 适合什么样的用户？

Loomy 适合需要频繁处理信息整理、内容创作、文档协作、任务跟进和跨工具执行的人群。比如：
*   自媒体运营
*   办公室白领
*   内容团队
*   小型业务团队
*   电商从业者
*   希望把重复流程交给 AI 协助完成的个人用户

如果你的工作中经常需要在聊天、邮件、文档、网页和待办之间反复切换，Loomy 会更容易发挥价值。
