# 快速开始

## 加入 Waiting List
首先在官网申请加入 Waiting List，提交基础信息后等待审核。

![加入 Waiting List](/loomy/quick-start/waiting.png)

## 获取邀请码
申请通过后，你会获得专属邀请码。邀请码是当前登录和体验 Loomy 的必要凭证，请妥善保存。

![获取邀请码](/loomy/quick-start/qmail.png)

进入 Loomy 官网，选择相应平台，点击下载按钮，即可下载 Loomy 客户端。

![下载 Loomy](/loomy/quick-start/download.png)

## 登录 Loomy
打开 Loomy 后，输入邀请码即可完成登录并开始使用。

![登录 Loomy](/loomy/quick-start/login.png)

## 开始体验
登录成功后，你就可以体验 Loomy 的核心能力，包括 AI 对话、任务管理、工作目录切换以及后续逐步开放的更多协作能力。
