# Frequently Asked Questions

## What are the conditions for one-click deployment?
You need to subscribe to the Basic or Professional plan. Unsubscribed users will be guided to upgrade when they click deploy.

## What should I do if AstronClaw is not responding?
*   Refresh the page
*   Go to settings and click **Restart AstronClaw**
*   If it still doesn't work, try auto-repair in the settings

## What is the difference between restart, reset, and delete?

| Operation | Effect |
| :--- | :--- |
| **Restart** | Keeps data, clears cache (similar to restarting a computer) |
| **Reset** | Deletes history and skills, restores default settings (similar to reinstalling the OS) |
| **Delete** | Completely deletes AstronClaw and all data (irrecoverable) |

## What file uploads are supported?
*   **Supported Formats**: pdf, doc, txt, jpg, etc., unlimited types
*   **Normal Files**: Each file size should not exceed 50MB, total file size should not exceed 250MB, number of files should not exceed 5
*   **Audio Files**: Not exceeding 100MB, total file size not exceeding 500MB, number of files not exceeding 5
*   **Image Files**: Not exceeding 20MB, total file size not exceeding 100MB, number of files not exceeding 5

## How to switch the chat model?
Enter the **Settings** page and switch the chat model in the model selection area.

## Can AstronClaw send files to me?
Currently, AstronClaw can receive files you upload and process them with tools, but it cannot directly send files to you in the interface yet. This feature will be launched in future versions. At this stage, you can access the generated files through other configured channels.
