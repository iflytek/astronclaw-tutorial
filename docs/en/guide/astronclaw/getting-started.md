# Getting Started

With just a few simple steps, you can have your own 24/7 online cloud digital lobster.

## Step 1: Deploy Now

Go to the **AstronClaw** page and click the `Deploy Now` button. The system will automatically create an AstronClaw for you.

## Step 2: Start Chatting

After deployment is complete, enter the chat home page:
*   **Welcome Message**: Hi, I am AstronClaw!
*   **File Upload**:
    *   Supports formats like `pdf/doc/txt/jpg`
    *   File size should not exceed 50MB, total file size should not exceed 250MB, and the number of files should not exceed 5
    *   Audio file size should not exceed 100MB, total audio file size should not exceed 500MB, and the number of files should not exceed 5
    *   Image size should not exceed 20MB, total image size should not exceed 100MB, and the number of files should not exceed 5

## Step 3: Install Skills

Enter the **Skills Market** to browse and install official preset skills:
*   Supports fuzzy name search
*   Click install to configure and use with one click

*(For more detailed introduction about skills, please refer to [Detailed Explanation of Skill System](./skills.md))*

## Step 4: Configure Message Channels

Enter the **Channel Configuration** page to connect to enterprise collaboration platforms. AstronClaw now supports:
*   Feishu Channel
*   DingTalk Channel
*   WeCom Channel

With just a simple configuration of the relevant App ID and credentials, your digital lobster can be on standby at any time in these chat softwares!
